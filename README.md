## A knock-knock like socket server for appreciating (unfunny) jokes

## Use instructions:

### Clone the repo:
1)  `git clone https://github.com/OmarMesqq/socketcom` 
2) `cd nowkgo` 

### Setup the server
1) `cd server`
2) Compile it: `go build -o server` 
3) Run it: `./server`

### Setup the client
1) `cd client` 
2) Compile it: `gcc -o client client.c`
3) Run it: `./client`
